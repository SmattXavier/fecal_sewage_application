# üö∞ Faecal Sludge Management (FSM) System Demo

A digital platform designed to streamline the management of sanitation services such as desludging operations, public toilet monitoring, incident reporting, and regulatory compliance. The system enables seamless communication between field operators, utilities, administrators, and customers, ensuring efficient faecal sludge collection, transport, treatment, and disposal. This project was initiated by the  Imo State Water and Sewerage Corporation (ISWSC), Imo State, Nigeria.

# Features
-- User Authentication
-- Incident Reporting & Response
-- Sludge collection volume.
-- Number of desludging operations.
-- Map visualizations.
-- GPS coordinates integration.
-- Request creation for desludging services.
-- View operational zones on an interactive GIS map.

This are the base features for the demo.
NOTE: This is not the repository for the project itself. Just a prototype done to land the contract.
